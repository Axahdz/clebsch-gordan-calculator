# Clebsch-Gordan Coefficient Calculator

This project is a desktop application developed in C# using WPF for calculating Clebsch-Gordan coefficients used in quantum mechanics.

## Overview

The application implements the mathematical logic required to compute coefficients and provides a user-friendly interface for input validation and result visualization.

## Features

- Calculation of Clebsch-Gordan coefficients
- Input validation
- Graphical user interface (WPF)
- Mathematical computation logic

## Technologies

- C#
- WPF (.NET)

## Use Case

This tool was designed to support quantum mechanics calculations, particularly in academic and research contexts.

## Getting Started

1. Open the project in Visual Studio
2. Build and run the application
